# Rodent-AMQP
Pure Java AMQP Messaging Middleware for fun and frolics :)

A clean room implementation of AMQP 1.0 

https://www.amqp.org/

To build:

gradle clean test

# The story so far ...

1. Basic Spring based framework built
2. Test frameworks added
3. Basic message decoder implemented for AMQP
